| 設定鍵      | 型別 / 範例                                | 功能說明                      | 建議                                        |
| -------- | -------------------------------------- | ------------------------- | ----------------------------------------- |
| `mode`   | `'css'`, `'legacy'`, `'avoid-all'` 或陣列 | 控制採用哪些分頁規則。可多選。           | 常用 `['css', 'legacy']`，必要時再加 `avoid-all`。 |
| `before` | CSS 選擇器字串 / 陣列，如 `'.break-before'`     | 在符合 selector 的元素「前面」強制換頁。 | 用在章節標題前。                                  |
| `after`  | CSS 選擇器，如 `'.section-end'`             | 在符合 selector 的元素「後面」強制換頁。 | 用在表格、段落結束後。                               |
| `avoid`  | CSS 選擇器，如 `'tr'`, `'.no-split'`        | 嘗試避免在符合 selector 的元素中間斷頁。 | 常用於 `<tr>`、卡片、圖片。                         |
